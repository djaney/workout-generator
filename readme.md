# Workout generator
## Install

    python3 -m pip install -e git+git@github.com:djaney/workout-generator.git#egg=workout-gen
    
## Run
    generate-workout [--help]
    
## Output

    Plyometric Push-up x 15
    Squat Jumps x 15
    Superman x 15
    Burpees x 15

    Do 5 sets of each exercise
